## 📚 Units Covered

### 📘 Unit 1: Basic OOP Concepts (9 Programs)
**Focus**: Fundamental Object-Oriented Programming principles

1. **Hospital Management System** - Classes, objects, encapsulation
2. **Bank Account System** - Encapsulation with private data members
3. **Employee Payroll System** - Default and parameterized constructors
4. **Vehicle Rental System** - Inheritance with Car and Bike classes
5. **Shape Drawing Application** - Polymorphism with Shape base class
6. **College Admission System** - Static vs instance members
7. **Student Marks Processing** - Parameter passing mechanism
8. **Result Processing System** - ArrayList with autoboxing/unboxing
9. **CPU Simulation** - Inner classes with ALU and Cache

### 📗 Unit 2: Advanced OOP & Collections (14 Programs)
**Focus**: Advanced Java features and Collections Framework

1. **Company Employee System** - Method overriding with super
2. **College System** - Super constructor chaining
3. **Shape System** - Runtime polymorphism with Shape array
4. **Banking System** - Final variables and methods
5. **Abstract Vehicle System** - Abstract classes and methods
6. **Smart Device** - Multiple interfaces (Callable & Camera)
7. **Bank Interface** - Interface constants and implementation
8. **Calculator** - Method overloading and overriding
9. **Package Access Control** - Public, private, protected access
10. **Array vs ArrayList** - Dynamic collection operations
11. **Vector Operations** - Vector class methods demonstration
12. **Garbage Collection** - finalize() and System.gc()
13. **StringTokenizer** - String tokenization techniques
14. **Advanced Hospital System** - Abstract class + interface + ArrayList

### 📙 Unit 3: Exception Handling (8 Programs)
**Focus**: Comprehensive exception handling strategies

1. **Banking Division by Zero** - ArithmeticException handling
2. **Array Index Out of Bounds** - Multiple catch blocks
3. **Hospital Age Validation** - IllegalArgumentException
4. **Login System** - Custom InvalidLoginException
5. **Password Validation** - Custom InvalidPasswordException
6. **Student Grading System** - Custom InvalidMarksException
7. **Banking Withdrawal System** - Custom InsufficientBalanceException
8. **File I/O Exception Handling** - FileNotFoundException, IOException

## 🚀 Getting Started

### 1. Clone or Download the Project
```bash
git clone <repository-url>
cd Scenario_Based_Questions
```

### 2. Compile the Project
```bash
# Navigate to src directory
cd src

# Compile all Java files
javac *.java */*.java */*/*.java */*/*/*.java

# Or use your IDE's build system
```

### 3. Run the Application
```bash
# Run the main class
java Main

# Or run from your IDE
```

### 4. Navigate the Menu
- Choose a unit (1-3)
- Select a specific program
- Follow on-screen instructions
- Press Enter to continue between programs

## 📖 Detailed Program Descriptions

### 🏥 Unit 1 Highlights

- Car and Bike derived classes with specialized behavior
